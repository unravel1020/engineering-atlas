Date: 2026-06-17

## Goal

将腾讯云 Ubuntu 服务器接入 VS Code Remote SSH。

## Environment

Local:
- Windows 11

Remote:
- Ubuntu 22.04

## Process

1. SSH 登录成功
2. 安装 Remote SSH 插件
3. VS Code Server 自动安装
4. 连接成功
